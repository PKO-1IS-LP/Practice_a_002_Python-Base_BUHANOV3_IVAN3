## Введение в tkinter

> 📌 **Важно!**  
> Этой библиотеке посвящено мало внимания, и найти в рунете курс, книгу или FAQ по ней довольно-таки сложно.

### Что такое tkinter?

**Tkinter** — это графическая библиотека, позволяющая создавать программы с оконным интерфейсом на Python.

> 💡 **Интересный факт**  
> Библиотека портирована из популярного на UNIX-системах инструментария Tk.

> 🏷️ **Термин**  
> **Виджеты** — стандартизированные компоненты графического пользовательского интерфейса, с которыми взаимодействует пользователь.

### Импорт библиотеки

```python
from tkinter import *
```

> ⚠️ **Внимание!**  
> В Python 2 нужно писать `Tkinter` с большой буквы, в Python 3 — `tkinter` с маленькой.

### Создание главного окна

```python
root = Tk()  # создание главного окна
root.title('Моя программа')  # заголовок окна
root.geometry('400x300')  # размер окна (ширина x высота)
root.mainloop()  # главный цикл обработки событий
```

> 🎯 **Ключевой момент**  
> `mainloop()` запускает бесконечный цикл обработки событий. Без этой команды окно мгновенно закроется!

> 📝 **Заметка**  
> `root` — это общепринятое название для главного окна, но вы можете назвать его как угодно.

## Основные виджеты

### Button (Кнопка)

```python
button1 = Button(root, text='ok', width=25, height=5, 
                 bg='black', fg='red', font='arial 14')
button1.pack()
```

> 🔍 **Детали**  
> `width` и `height` задаются в символах, не в пикселях!

> 🎨 **Цвета**  
> - `bg` (background) — цвет фона  
> - `fg` (foreground) — цвет текста

### Label (Метка)

```python
label1 = Label(root, text='Привет!', font='Arial 20', fg='blue')
label1.pack()
```

> ℹ️ **Для информации**  
> `Label` используется только для отображения текста, пользователь не может его редактировать.

### Entry (Поле ввода)

```python
entry1 = Entry(root, width=30, bd=5)  # bd - borderwidth (ширина границы)
entry1.pack()
```

> 📌 **Совет**  
> `bd` (borderwidth) увеличивает границу поля, делая его более заметным.

### Text (Многострочное поле)

```python
text1 = Text(root, height=7, width=7, font='Arial 14', wrap=WORD)
text1.pack()
```

> 🔄 **Режимы переноса**  
> - `WORD` — перенос по словам  
> - `CHAR` — перенос по символам  
> - `NONE` — без переноса

### Listbox (Список)

```python
listbox1 = Listbox(root, height=5, width=15, selectmode=EXTENDED)
```

> ✅ **Режимы выбора**  
> - `SINGLE` — только один пункт  
> - `EXTENDED` — несколько пунктов (с Ctrl/Shift)  
> - `BROWSE` — выбор движением мыши  
> - `MULTIPLE` — несколько пунктов простым кликом

### Frame (Рамка)

```python
frame1 = Frame(root, width=100, height=100, bg='green', bd=5)
```

> 🖼️ **Для чего нужен Frame**  
> Контейнер для группировки виджетов. Помогает лучше организовать интерфейс.

### Checkbutton (Флажок)

```python
var1 = IntVar()  # специальная переменная
check1 = Checkbutton(root, text='1 пункт', variable=var1, onvalue=1, offvalue=0)
```

> 🔢 **Типы переменных tkinter**  
> - `IntVar()` — для целых чисел  
> - `StringVar()` — для строк  
> - `BooleanVar()` — для булевых значений  
> - `DoubleVar()` — для чисел с плавающей точкой

### Radiobutton (Переключатель)

```python
var = IntVar()
rbutton1 = Radiobutton(root, text='1', variable=var, value=1)
```

> ⚡ **Важно!**  
> Все радиокнопки в группе используют **одну переменную**. Только одна кнопка может быть выбрана.

## Упаковщики (менеджеры геометрии)

> 📦 **Три способа упаковки**  
> В tkinter есть три упаковщика: `pack()`, `place()`, `grid()`. Нельзя смешивать их для одного родителя!

### pack()

```python
button1.pack(side='left')  # разместить слева
```

> 🧭 **Параметры side**  
> - `'left'` — слева  
> - `'right'` — справа  
> - `'top'` — сверху (по умолчанию)  
> - `'bottom'` — снизу

### place()

```python
button1.place(x=100, y=50, width=200, height=30)
```

> 🎯 **Точное позиционирование**  
> `place()` даёт полный контроль, но не адаптируется к изменению размера окна.

### grid()

```python
entry1.grid(row=0, column=0, columnspan=3)
```

> 📊 **Сетка**  
> - `row` — номер строки  
> - `column` — номер столбца  
> - `columnspan` — сколько столбцов занимает виджет  
> - `sticky` — "приклеивание" к стороне (`'n'`, `'s'`, `'w'`, `'e'`)

## Привязка событий

```python
виджет.bind('<событие>', функция_обработчик)
```

> 🖱️ **События мыши**

| Событие | Что делает |
|---------|------------|
| `<Button-1>` | Левая кнопка мыши |
| `<Button-2>` | Средняя кнопка |
| `<Button-3>` | Правая кнопка |
| `<Double-Button-1>` | Двойной клик |
| `<Motion>` | Движение мыши |

> ⌨️ **События клавиатуры**

| Событие | Что делает |
|---------|------------|
| `<Return>` | Enter |
| `<Control-c>` | Ctrl+C |
| `<F1>` | F1 |
| `<a>` | Клавиша A |

### Функция-обработчик

```python
def leftclick(event):
    print(f'Координаты: {event.x}, {event.y}')
```

> 🎁 **Атрибуты event**  
> - `event.x`, `event.y` — координаты мыши  
> - `event.widget` — виджет, на котором произошло событие  
> - `event.keysym` — название нажатой клавиши

## Продвинутые виджеты

### Scale (Шкала/Ползунок)

```python
scale = Scale(root, orient=HORIZONTAL, length=300, 
              from_=50, to=80, tickinterval=5, resolution=5)
```

> ⚙️ **Параметры Scale**  
> - `from_` — минимальное значение (с подчёркиванием!)  
> - `tickinterval` — интервал между метками  
> - `resolution` — шаг перемещения

### Scrollbar (Полоса прокрутки)

```python
scr1 = Scrollbar(root, command=text1.yview)
text1.configure(yscrollcommand=scr1.set)
```

> 🔄 **Двусторонняя связь**  
> Скроллбар управляет текстом, а текст управляет скроллбаром!

## Стикеры и заметки

> ⭐ **Лучшие практики**

- ✅ Всегда используйте `mainloop()` в конце программы
- ✅ Давайте виджетам осмысленные имена
- ✅ Используйте `grid()` для сложных интерфейсов
- ✅ Не забывайте про отступы (`padx`, `pady`)

> ⚠️ **Частые ошибки**

- ❌ Забыли `mainloop()` — окно не появится
- ❌ Смешиваете `pack()` и `grid()` для одного родителя
- ❌ Не импортировали `messagebox` для диалогов
- ❌ Используете `from` вместо `from_` в Scale

> 💡 **Полезные советы**

| Проблема | Решение |
|----------|---------|
| Русские буквы | В Python 3 всё работает, в Python 2 нужен префикс `u` |
| Прозрачность | `root.attributes('-alpha', 0.9)` |
| Поверх всех окон | `root.attributes('-topmost', True)` |
| Иконка | `root.iconbitmap('icon.ico')` |

> 🚀 **Для быстрого старта**

```python
# Минимальная программа
from tkinter import *
root = Tk()
Label(root, text='Привет, мир!').pack()
root.mainloop()
```

## Примеры программ с комментариями

### Пример 1: Калькулятор (со стикерами)

```python
from tkinter import *

def calculate():
    """Функция вычисления результата"""
    try:
        num1 = float(entry1.get())
        num2 = float(entry2.get())
        operation = operation_var.get()
        
        # Выполняем операцию
        if operation == '+':
            result = num1 + num2
        elif operation == '-':
            result = num1 - num2
        elif operation == '*':
            result = num1 * num2
        elif operation == '/':
            if num2 == 0:
                result = 'Ошибка: деление на 0'
            else:
                result = num1 / num2
        else:
            result = 'Выберите операцию'
        
        label_result.config(text=f'Результат: {result}')
    except ValueError:
        label_result.config(text='Ошибка: введите числа')

# Создание главного окна
root = Tk()
root.title('Калькулятор')  # Заголовок окна
root.geometry('300x250')    # Размер окна
root.resizable(False, False)  # Запрещаем изменение размера

# Поля ввода
Label(root, text='Число 1:').grid(row=0, column=0, padx=5, pady=5)
entry1 = Entry(root, width=15)
entry1.grid(row=0, column=1, padx=5, pady=5)

Label(root, text='Число 2:').grid(row=1, column=0, padx=5, pady=5)
entry2 = Entry(root, width=15)
entry2.grid(row=1, column=1, padx=5, pady=5)

# Выбор операции
Label(root, text='Операция:').grid(row=2, column=0, padx=5, pady=5)
operation_var = StringVar()
operation_var.set('+')  # Операция по умолчанию

operations = ['+', '-', '*', '/']
for i, op in enumerate(operations):
    Radiobutton(root, text=op, variable=operation_var, 
                value=op).grid(row=2, column=i+1, padx=2)

# Кнопка вычисления
Button(root, text='Вычислить', command=calculate).grid(
    row=3, column=0, columnspan=4, pady=10)

# Результат
label_result = Label(root, text='Результат:', font='Arial 12 bold')
label_result.grid(row=4, column=0, columnspan=4, pady=10)

root.mainloop()
```

> 📌 **Комментарии к коду**  
> - `try/except` обрабатывает ошибки ввода  
> - `columnspan=4` растягивает кнопку на 4 колонки  
> - `resizable(False, False)` фиксирует размер окна

## №1: Моё яркое приветствие

```python
import tkinter as tk
# Импортируем библиотеку tkinter и даем ей короткое имя 'tk'
# tkinter - это стандартная библиотека Python для создания графических интерфейсов

window = tk.Tk()
# Создаем главное окно программы
# Tk() - это класс, который создает базовое окно приложения

window.title("Моё яркое приветствие")
# Устанавливаем заголовок окна
# title() - метод, который меняет текст в верхней части окна

window.geometry("500x200")
# Устанавливаем размер окна: ширина 500 пикселей, высота 200
# geometry() - метод для задания размеров и положения окна

window.configure(bg="#FFFACD")
# Настраиваем фон окна
# bg (background) - параметр цвета фона
# "#FFFACD" - HEX-код светло-желтого цвета

label1 = tk.Label(
    # Создаем первый текстовый элемент (метку)
    # Label - это виджет для отображения текста
    window,
    # Первый аргумент - родительский элемент (главное окно)
    
    text="HELLO WORLD!",
    # Текст, который будет отображаться на метке
    
    font=("Impact", 48, "bold"),
    # Настройка шрифта: название, размер, стиль
    # "bold" - жирное начертание
    
    fg="#800080",
    # fg (foreground) - цвет текста
    # "#800080" - фиолетовый цвет
    
    bg="#FFFACD"
    # bg (background) - цвет фона метки
    # Ставим такой же, как у окна, чтобы не было видно границ
)
# Закрываем создание первого Label

label1.pack(expand=True)
# Размещаем первую метку в окне
# pack() - метод компоновки (упаковки)
# expand=True - позволяет метке расширяться и центрироваться

label2 = tk.Label(
    # Создаем второй текстовый элемент
    window,
    # Родитель - главное окно
    
    text="Это мой первый яркий текст",
    # Текст второй метки
    
    font=("Comic Sans MS", 24, "italic"),
    # Шрифт: Comic Sans MS, размер 24, курсив ("italic")
    
    fg="#C71585",
    # "#C71585" - темно-розовый цвет (медиум-виолет-ред)
    
    bg="#FFFACD"
    # Тот же желтый фон, что и у окна
)

label2.pack(expand=True)
# Размещаем вторую метку под первой
# expand=True - тоже центрируется

window.mainloop()
# Запускаем главный цикл обработки событий
# Без этой строки окно бы появилось и сразу закрылось
# mainloop() держит окно открытым и обрабатывает действия пользователя
```

## №12: Три блока

```python
import tkinter as tk
# Импортирует библиотеку tkinter и дает ей псевдоним 'tk' для краткости.
# Tkinter - стандартная библиотека Python для создания графического интерфейса.

root = tk.Tk()
# Создает главное окно приложения. root - это корневой виджет,
# который будет содержать все остальные элементы интерфейса.

root.title("Три блока")
# Устанавливает заголовок главного окна. Текст "Три блока"
# появится в заголовке окна.

root.geometry("600x300")
# Задает размер главного окна: ширина 600 пикселей, высота 300 пикселей.
# Формат строки: "ширина x высота".

root.configure(bg='white')
# Настраивает (configure) главное окно. Устанавливает белый ('white')
# фоновый цвет (bg = background) для корневого окна.

blocks = [
    # Создает список словарей. Каждый словарь содержит параметры для одного блока:
    {"bg": "#FF4040", "text": "Красный", "fg": "white"},
    # bg: цвет фона (красный в HEX-формате)
    # text: текст, который будет отображаться
    # fg: цвет текста (foreground = белый)
    
    {"bg": "#4040FF", "text": "Синий", "fg": "white"},
    # Второй блок с синим фоном
    
    {"bg": "#40FF40", "text": "Зелёный", "fg": "white"}
    # Третий блок с зеленым фоном
]

for block in blocks:
    # Начинает цикл for, который пройдет по каждому элементу списка blocks.
    # На каждой итерации переменная block будет содержать очередной словарь.
    
    frame = tk.Frame(
        root,
        # Создает фрейм (контейнер) внутри главного окна (root)
        width=180,
        # Устанавливает ширину фрейма 180 пикселей
        height=260,
        # Устанавливает высоту фрейма 260 пикселей
        bg=block["bg"]
        # Устанавливает цвет фона фрейма из значения ключа "bg" текущего словаря block
    )
    
    frame.pack(side="left", padx=0, pady=0)
    # Размещает фрейм в окне с помощью менеджера геометрии pack.
    # side="left" - размещает фреймы слева направо (по порядку)
    # padx=0, pady=0 - отступы по горизонтали и вертикали равны 0
    
    frame.pack_propagate(False)
    # Запрещает фрейму изменять свой размер под содержимое.
    # Благодаря этому фрейм сохранит размеры 180x260,
    # даже если содержимое меньше или больше.
    
    label = tk.Label(
        frame,
        # Создает текстовую метку (label) внутри текущего фрейма
        text=block["text"],
        # Текст метки берется из ключа "text" текущего словаря
        font=("Arial", 28, "bold"),
        # Шрифт: Arial, размер 28, жирный (bold)
        bg=block["bg"],
        # Цвет фона метки такой же, как у фрейма (из ключа "bg")
        fg=block["fg"]
        # Цвет текста берется из ключа "fg" текущего словаря
    )
    
    label.pack(expand=True)
    # Размещает метку внутри фрейма с помощью pack.
    # expand=True позволяет метке расширяться и занимать все доступное
    # пространство внутри фрейма, центрируя текст.

root.mainloop()
# Запускает главный цикл обработки событий tkinter.
# Программа будет ожидать действий пользователя и обновлять окно,
```

```python
# Настраивает (configure) главное окно. Устанавливает очень светло-серый цвет фона
# (#F5F5F5) для корневого окна.

colors = [
    # Создает двумерный список (матрицу) цветов 3x4
    # Каждый внутренний список - строка палитры с 4 цветами в HEX-формате
    ["#FF6B6B", "#4ECDC4", "#45B7D1", "#96CEB4"],
    # Первая строка: красноватый, бирюзовый, голубой, светло-зеленый
    
    ["#FFEEAD", "#D4A5A5", "#9B59B6", "#3498DB"],
    # Вторая строка: светло-желтый, розоватый, фиолетовый, синий
    
    ["#E74C3C", "#2ECC71", "#F1C40F", "#34495E"]
    # Третья строка: красный, зеленый, желтый, темно-синий
]

main_container = tk.Frame(root, bg='#F5F5F5')
# Создает главный контейнер (фрейм) внутри корневого окна.
# bg='#F5F5F5' - цвет фона такой же, как у главного окна.
# Этот фрейм будет содержать все элементы палитры.

main_container.pack(expand=True)
# Размещает главный контейнер с помощью pack.
# expand=True позволяет контейнеру расширяться и занимать все доступное пространство,
# что помогает центрировать содержимое.

def get_text_color(bg_color):
    # Определяет функцию, которая определяет цвет текста (черный или белый)
    # на основе яркости фонового цвета для лучшей читаемости
    
    bg_color = bg_color.lstrip('#')
    # Удаляет символ '#' из начала строки, оставляя только HEX-значение цвета
    # Например, "#FF6B6B" -> "FF6B6B"
    
    r, g, b = int(bg_color[0:2], 16), int(bg_color[2:4], 16), int(bg_color[4:6], 16)
    # Преобразует HEX-компоненты цвета в десятичные числа:
    # bg_color[0:2] - первые два символа (красный) -> int(..., 16) преобразует из 16-ричной в 10-чную
    # bg_color[2:4] - следующие два символа (зеленый)
    # bg_color[4:6] - последние два символа (синий)
    # Например, для "FF6B6B": r=255, g=107, b=107
    
    brightness = (r * 299 + g * 587 + b * 114) / 1000
    # Вычисляет яркость цвета по формуле восприятия человеческим глазом
    # Коэффициенты (299, 587, 114) учитывают, что глаз по-разному чувствителен к разным цветам
    # (зеленый воспринимается как самый яркий)
    # Результат от 0 (черный) до 255 (белый)
    
    return "#000000" if brightness > 128 else "#FFFFFF"
    # Возвращает черный цвет (#000000) для светлых фонов (яркость > 128)
    # Или белый цвет (#FFFFFF) для темных фонов (яркость <= 128)
    # Это обеспечивает контрастность текста на любом фоне

for row in range(3):

    # bg='#F5F5F5' - фон такой же, как у главного контейнера
    
    row_frame.pack()
    # Размещает фрейм строки. По умолчанию pack располагает элементы вертикально,
    # поэтому строки будут идти сверху вниз
    
    for col in range(4):
        # Внутренний цикл для создания 4 столбцов в текущей строке (col от 0 до 3)
        
        color = colors[row][col]
        # Получает цвет из двумерного списка colors по индексам [row][col]
        
        text_color = get_text_color(color)
        # Вызывает функцию get_text_color для определения цвета текста
        # (черный или белый) на основе яркости фонового цвета
        
        color_frame = tk.Frame(
            row_frame,
            # Создает фрейм для отдельного цвета внутри row_frame
            width=100,
            # Ширина фрейма 100 пикселей
            height=80,
            # Высота фрейма 80 пикселей
            bg=color,
            # Цвет фона - текущий цвет из палитры
            bd=1,
            # Толщина границы (border) = 1 пиксель
            relief='solid',
            # Стиль границы - сплошная линия
            highlightbackground='#CCCCCC'
            # Цвет подсветки границы - светло-серый
        )
        color_frame.pack(side='left', padx=1, pady=1)
        # Размещает фрейм цвета с помощью pack.
        # side='left' - размещает фреймы слева направо (горизонтально в строке)
        # padx=1, pady=1 - добавляет отступы в 1 пиксель между ячейками

            color_frame,

            # .upper() преобразует все буквы в заглавные
            font=("Arial", 12),
            # Шрифт: Arial, размер 12
            bg=color,
            # Цвет фона метки такой же, как у фрейма (текущий цвет)
            fg=text_color
            # Цвет текста - определенный функцией (черный или белый)
        )
        hex_label.pack(expand=True)
        # Размещает метку внутри color_frame.
        # expand=True позволяет метке расширяться и занимать все доступное
        # пространство внутри фрейма, центрируя текст
```

```python
    # side='left' - метка будет слева в строке

    entry = tk.Entry(
        row_frame,
        # Создает поле ввода внутри row_frame

    entries[label_text] = entry

button_container = tk.Frame(main_container, bg='#ECF0F1')

save_button_frame = tk.Frame(
```

## Часть 1: Целочисленная арифметика (Операторы `//` и `%`)

Базовые операции целочисленного деления (`//`) и взятия остатка (`%`) являются основой для решения широкого класса задач, связанных с "цифровой" арифметикой.

### Задача 1: Последняя цифра числа
*   **Условие:** Дано натуральное число \( N \). Найти его последнюю цифру.
*   **Решение:** Остаток от деления на 10 всегда дает последнюю цифру числа в десятичной системе счисления.
*   **Код:**
    ```python
    n = int(input())
    print(n % 10)
    ```

### Задача 2: Первая цифра двузначного числа
*   **Условие:** Дано двузначное число. Найти число его десятков.
*   **Решение:** Для двузначного числа целочисленное деление на 10 отсекает единицы, оставляя десятки.
*   **Код:**
    ```python
    n = int(input())
    print(n // 10)
    ```

### Задача 3: Число десятков (предпоследняя цифра)
*   **Условие:** Дано целое неотрицательное число \( N \). Найти его предпоследнюю цифру (число десятков). Если число однозначное, вывести 0.
*   **Решение:** Сначала убираем последнюю цифру (`N // 10`), а затем берем последнюю цифру у получившегося числа (`% 10`).
*   **Код:**
    ```python
    n = int(input())
    print((n // 10) % 10)
    ```

### Задача 4: Сумма цифр трехзначного числа
*   **Условие:** Дано трехзначное число. Найти сумму его цифр.
*   **Решение:** Извлекаем каждую цифру с помощью операторов `//` и `%` и складываем их.
*   **Код:**
    ```python
    n = int(input())
    hundreds = n // 100
    tens = (n // 10) % 10
    units = n % 10
    print(hundreds + tens + units)
    ```

### Задача 5: Покупка пирожков
*   **Условие:** Пирожок стоит \( A \) рублей \( B \) копеек. Найти стоимость \( N \) пирожков в рублях и копейках.
*   **Решение:** Переводим цену в копейки, умножаем на \( N \), затем конвертируем обратно в рубли и копейки.
*   **Код:**
    ```python
    a = int(input())
    b = int(input())
    n = int(input())
    total_kopecks = (a * 100 + b) * n
    print(total_kopecks // 100, total_kopecks % 100)
    ```

## Часть 2: Условные операторы

Условные операторы (`if`, `elif`, `else`) позволяют программе принимать решения в зависимости от входных данных.

### Задача 6: Максимум из двух чисел
*   **Условие:** Даны два числа. Вывести наибольшее. Если числа равны, вывести любое из них.
*   **Решение:** Сравниваем числа с помощью `if-else`.
*   **Код (Вариант 1 - Классический):**
    ```python
    a, b = int(input()), int(input())
    if a > b:
        print(a)
    else:
        print(b)
    ```
*   **Код (Вариант 2 - Тернарный оператор):**
    ```python
    a, b = int(input()), int(input())
    res = a if a > b else b
    print(res)
    ```

### Задача 7: Существование треугольника
*   **Условие:** Даны три натуральных числа — длины сторон. Проверить, существует ли треугольник с такими сторонами.
*   **Решение:** Треугольник существует тогда и только тогда, когда каждая сторона меньше суммы двух других (неравенство треугольника).
*   **Код:**
    ```python
    a, b, c = int(input()), int(input()), int(input())
    if a < b + c and b < a + c and c < a + b:
        print("YES")
    else:
        print("NO")
    ```

### Задача 8: Наименьший делитель
*   **Условие:** Дано целое число \( N \ge 2 \). Найти его наименьший натуральный делитель, отличный от 1.
*   **Решение:** Перебираем числа от 2 до \( \sqrt{N} \). Первый же найденный делитель будет наименьшим. Если делитель не найден, число простое, и ответ — само \( N \).
*   **Код:**
    ```python
    n = int(input())
    d = 2
    while d * d <= n:
        if n % d == 0:
            print(d)
            break
        d += 1
    else:  # сработает, если break не был выполнен
        print(n)
    ```

## Часть 3: Циклические алгоритмы

Циклы (`while`, `for`) используются для многократного выполнения повторяющихся действий.

### Задача 9: Парты
*   **Условие:** В трех классах учатся \( A \), \( B \) и \( C \) учеников. За одной партой могут сидеть двое. Найти минимальное количество парт для всех классов.
*   **Решение:** Для каждого класса округляем количество учеников вверх до ближайшего четного и делим на 2. Удобная формула: `(ученики + 1) // 2`.
*   **Код:**
    ```python
    a, b, c = int(input()), int(input()), int(input())
    desks = (a + 1) // 2 + (b + 1) // 2 + (c + 1) // 2
    print(desks)
    ```

### Задача 10: Следующее четное
*   **Условие:** Дано натуральное \( N \). Вывести следующее за ним четное число.
*   **Решение:** Можно использовать формулу, основанную на целочисленном делении: `(N // 2) * 2 + 2`.
*   **Код:**
    ```python
    n = int(input())
    print((n // 2) * 2 + 2)
    ```

### Задача 11: Степени двойки
*   **Условие:** Дано натуральное \( N \). Вывести все степени двойки, не превосходящие \( N \), в порядке возрастания.
*   **Решение:** Заводим переменную-степень, равную 1. В цикле печатаем её, пока она не станет больше \( N \), каждый раз умножая на 2.
*   **Код:**
    ```python
    n = int(input())
    p = 1
    while p <= n:
        print(p, end=" ")
        p *= 2
    ```

### Задача 12: Проверка на степень двойки
*   **Условие:** Дано натуральное \( N \). Проверить, является ли оно точной степенью двойки.
*   **Решение (Битовый способ):** У степени двойки в двоичной записи только одна единица. Следовательно, побитовое И числа и предшествующего ему числа всегда равно нулю: `(N & (N-1)) == 0`.
*   **Код:**
    ```python
    n = int(input())
    if n > 0 and (n & (n - 1)) == 0:
        print("YES")
    else:
        print("NO")
    ```

### Задача 13: Дробная часть числа
*   **Условие:** Дано положительное действительное число \( X \). Вывести его дробную часть.
*   **Решение:** Дробная часть равна разности между самим числом и его целой частью.
